# Teams Tab App Manifest

## Setup

1. `manifest.json` の `{{MICROSOFT-APP-ID}}` を Azure AD App ID に置換
2. `{{DEPLOYMENT-ID}}` を GAS デプロイ ID に置換
3. `color.png` (192x192) と `outline.png` (32x32) をこのディレクトリに追加
4. 3ファイル (manifest.json, color.png, outline.png) を ZIP にパッケージ
5. Teams Admin Center または Teams クライアントからサイドロード

## Icon Requirements

- `color.png`: 192x192px, full color
- `outline.png`: 32x32px, white with transparent background
